# employapply-parent

##### 入职审批工作流
  * api：后台的service和dao的接口
  * biz：后台service接口的实现
  * dao：后台dao接口的实现
  * entities：javaBean实体类
  * extjs：前台增删改查界面的展示
  * rest：前台到后台的rest服务配置以及工作流定义文件
  * web：前台菜单的实现

# 流程说明
  * 行政科研人员
    * 初试复试结束后->校领导审批->结束
  * 专职教师
    * 面试试讲结束后->分院领导审批->校领导审批->结束
  * 兼职教师
    * 试讲结束后->分院领导审批->结束

# 注意事项
  根据不同的人员类别，走不同的工作流支路
